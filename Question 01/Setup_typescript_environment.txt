Install Visual Studio Code: You can download Visual studio code from the official website:
(https://code.visualstudio.com/download)

Install Node.js: TypeScript needs Node.js to run. You can download and install Node.js from the official website:
(https://nodejs.org/en/download).

Install TypeScript: Once Node.js is installed, open a terminal (command prompt on Windows, terminal on macOS/Linux) and install TypeScript globally using npm (Node Package Manager) by running the following command:
(npm install -g typescript)
